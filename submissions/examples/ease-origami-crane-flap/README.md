# Origami Crane Flap

1. What does this do? Creates a floating, 3D paper crane with flapping wings built entirely using CSS borders and 3D transforms.
2. How is it used? Apply the `.crane-container` and its nested shapes to the desired element.
3. Why is it useful? It showcases advanced CSS 3D rendering and `transform-style: preserve-3d` capabilities, providing a beautiful floating animation without the need for SVG files or JavaScript.
