# Nixie Tube Clock Glow

1. What does this do? Simulates a vintage Nixie tube using multi-layered `text-shadow` for glowing gas effects and `backdrop-filter` for the glass enclosure.
2. How is it used? Apply the `.tube` container and its internal `.digit` layers to any numerical text.
3. Why is it useful? It creates an authentic, retro-futuristic digital display that flickers dynamically without requiring any JavaScript or SVG assets.
