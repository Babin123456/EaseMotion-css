# Steampunk Gear Mechanism

1. What does this do? Creates a system of interlocking, rotating gears with a metallic texture using advanced CSS gradients and animations.
2. How is it used? Add the `.mechanism-container` and its nested `.gear` classes to an element.
3. Why is it useful? It demonstrates how to use `repeating-conic-gradient` to procedurally generate complex shapes like gear teeth without relying on SVG or extra DOM elements.
