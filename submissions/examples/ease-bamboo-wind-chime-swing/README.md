# Bamboo Wind Chime Swing

1. What does this do? Simulates a hanging bamboo wind chime that sways organically as if blown by a gentle breeze.
2. How is it used? Place the `.chime-container` structure and its inner `.chime` elements in your HTML.
3. Why is it useful? It provides a serene, chaotic, and natural-feeling animation by combining varying `animation-duration` and `animation-delay` properties on multiple objects.
