# Retro Cassette Tape Spin

1. What does this do? Creates a CSS-only vintage cassette tape with spinning spools and a wobbling neon gradient label.
2. How is it used? Apply the `.cassette` container structure and its nested children to an element.
3. Why is it useful? It provides a nostalgic, highly detailed music-themed loading or display element without the overhead of SVG or JavaScript, aligning perfectly with EaseMotion's CSS-only philosophy.
