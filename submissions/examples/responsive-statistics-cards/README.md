# Responsive Statistics Cards

## Description

A responsive statistics card component built using only HTML and CSS. It is designed to display key metrics such as users, projects, revenue, success rate, downloads, or any other numerical information in a clean and modern layout.

## Features

- Responsive design
- Modern card layout
- Hover elevation effect
- Emoji/Icon support
- Lightweight and reusable
- Pure HTML & CSS
- Easy to customize

## Usage

```html
<div class="stats-container">

    <div class="stat-card">
        <div class="icon">👥</div>
        <h2>1.2K+</h2>
        <p>Users</p>
    </div>

</div>
```
