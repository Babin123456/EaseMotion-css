# CSS Elastic-Slide Hero Section

A modern **Elastic-Slide Hero Section** built using pure HTML5 and CSS3 for creative portfolio layouts. This component features an elastic entrance animation, floating visual cards, responsive layout, and accessibility support.

---

## ✨ Features

- Pure HTML5 & CSS3
- Elastic slide entrance animation
- Floating layered hero cards
- Interactive hover effects
- Responsive two-column layout
- CSS Custom Properties
- Modern gradient styling
- `prefers-reduced-motion` support
- No JavaScript required

## 🚀 Usage

1. Copy the component into your project.
2. Link the stylesheet:

```html

```

3. Open `demo.html` in any modern browser.
